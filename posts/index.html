开学这么久，这些有趣的方言你知道了吗？

开学这么久，这些有趣的方言你知道吗 开学这么长时间了，连最难返校的新疆友友也都陆陆续续返校了，在这一个多月和自己新室友相处的时候，不同地域的方言是否出现了有趣的碰撞呢？不如今天和V妮再复习一点有趣的方言吧~ 呼和浩特： 室友有个呼市人，那股类似东北又别具一格的感觉肯定可以抓住你的心 正说着普通话（其实带着一点点东北味）突然来一句：闹不吉米 嗯？什么鬼，吉米？多维奇？nonono 闹不吉米就是搞不懂的意思 “你这袋子还要不要了， 我给你帽了啊？”，帽？这可不是篮球场怎么能说帽就帽。 所以帽了就是丢掉的意思 正写着作业，抬头问一句：“欸？你第三题会写不”。“哦，那题我不带药写” 好了又来，但不带药啥意思。。。不带要就是不打算的意思 说别人是 nia ，说自己是 e 。下次面对内蒙人，你一定能理解了吧 云南： 说到云南，你脑子里都是什么？ 不对！虽然可能有一点对，不过往下看 “大家好，我的特长是抓球。”嗯？把球抓起来吗，当然不是，抓球就是踢球的意思 要想知道云南人可爱在哪？“我要出克”“你克哪咧”去就是克（从壳里爬出来一个云南小可爱） “欸？我的校园卡找不到啦！”（急忙忙翻找了一会）“哦~原来在裤包里”裤包=裤兜 最可爱的不过下面这两个： 睡觉：睡诺诺（nuō）吃饭：吃茫茫（māng） 河南： 如果你的室友是个河南考生，那你一定要珍惜他/她，毕竟为了来到你身边，他可能砍掉了一个省的人，所以就别指着下面的图问他这是不是河南通宝了 说到河南口音，可能大部分人脑子里都是“中！”（好） 但是实际情况可能更有趣： 你给河南人讲一个令人震惊的消息，他可能会说：“乖乖嘞，雀（quō）我类吧”（天哪，骗我的吧） 给他说一个厉害的人或物，他可能会说：“真不瓤”（真厉害）“真排场”（真好看） 他要是校园卡丢了：“卡找不桌了，可桌急”（很着急），卡又找到了：“怪得劲”（挺爽） 说头是“地脑”，说膝盖是“不老盖”，说睡觉是“谁（shèi）觉儿” 东北： 作为常年在春晚小品亮相的，东北口音在同化别人上从来没有输过，幽默的东北人总能不知不觉承包一群人的笑点 开学的时候，在每个寝室安插一个东北人，那想必一学期后满学校都是一样的口音： “必须——得！！” “你瞅啥？！” “瞅你咋地！！！！？？” 尾椎：尾（yǐ）巴 胯：胯骨轴子 膝盖：波愣盖儿 湖南： 湖南的友友性格豪爽，调子高，那一口塑料普通话，活生生就是在卖萌 如果你有幸捕捉到了一个湖南人，你一定要让他念这俩句话： 茉莉奶绿 六棵柳树扭六扭 如果他是个资深塑料普通话使用者，这大概能承包你一年的笑点。

崭新CentOS7虚拟机换源

近日整了一台CentOS虚拟机这里记录配置方法 Linux版本：CentOS7 软件：VMware Workstation Pro 配置网络 首先将虚拟机配置为桥接模式 首先ping一下127.0.0.1看看网卡是否是好的 这个时候ping百度看看通不通，如果通了直接换源，如果没通继续： 首先查看当前网络连接 nmcli connection show 记住网络设备： 设置DNS服务器 nmcli con mod ens33 ipv4.dns "114.114.114.114 8.8.8.8" nmcli con up ens33 此时再ping即可ping通 更换yum源 首先备份原来的CentOS官方yum源 cp /etc/yum.repos.d/CentOS-Base.repo /etc/yum.repos.d/CentOS-Base.repo.bak 获取阿里的yum源覆盖本地官方yum源 curl -o /etc/yum.repos.d/CentOS-Base.repo http://mirrors.aliyun.com/repo/Centos-7.repo 清理yum缓存，并生成新的缓存 yum clean all yum makecache 此时测试下载： yum install -y wget yum install -y vim 没报错，成功！

关于Github中无法加载图片

在配置了Github Pages之后，发现网站中所有图片都没有了，检查之后发现没有成功加载图片， 但是查看Github仓库之后发现路径并没有问题，但是图片就是显示404 即便是在仓库中显示这个文件是存在的，这里出现的问题就是文件命名规范的问题 如果使用Github为图床时，文件的路径中文件夹名称首不能出现字符 "_"（下划线），不然会出现文件访问不了的情况，这时只需要将文件夹名称中下划线删除，然后批量修改HTML页面即可 此时再访问网站页面即可发现问题已经解决~
