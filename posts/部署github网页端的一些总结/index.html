部署Github网页端的一些总结#

部署步骤#

  1. 首先，清空远程库（也可以不清空，但一定要备份！接下来的操作将会覆盖掉你的远程库）
  2. 在命令提示符下进入site的位置，输入命令hugo生成\public目录
  3. 进入public右键git bash here
  4. 输入命令：git init出现如图所示小标算成功
  5. 接着，将文档放入缓存区：git add -A或git add .
  6. 然后将所有文件提交至本地仓库：git commit -am"随便写点什么"
  7. 远程创建新分支：git remote add origin 你的仓库的url
  8. 将文件推送至远程库git push -f origin master
  9. 合并分支：git push --set-upstream origin master
  10. 在github上Setting中的Page界面更改域名
